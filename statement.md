# Welcome!
Ce code vous parait peut être compliqué mais c'est en réalité le programme le plus simple que l'on
peut écrire en C++. Je vous laisse cliquer sur run.


```C++ runnable
#include <iostream>

using namespace std;

int main() 
{
    cout << "Hello, World!";
    return 0;
}
```
